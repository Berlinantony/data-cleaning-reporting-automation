# Data Cleaning & Reporting Automation

## Project Overview

This project automates data cleaning and reporting processes using Python. It handles missing values, removes duplicates, standardizes data, and generates automated reports and visualizations.

## Features

* Load and process CSV datasets
* Detect and handle missing values
* Remove duplicate records
* Standardize text data
* Generate automated Excel reports
* Create visual summaries using Matplotlib
* Export cleaned datasets

## Technologies Used

* Python
* Pandas
* Matplotlib
* OpenPyXL

## Outputs Generated

* cleaned_data.xlsx
* data_report.xlsx
* report_chart.png
* sales_by_category.png
